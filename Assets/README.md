**a. Затраченное время:** 4 дня

**b. Описание реализации:**
Попробовал реализовать через архитектуру **MVC** плюс вспомогательные компоненты.  
На сцене можно найти объект **Controllers**, который содержит в себе главный контроллер с точкой входа и который создаёт все необходимые классы и инициализирует их.  
Также на этом GameObject находятся спавнеры для сетки и стеков.  
Есть **ScriptableObject**-объекты для настройки цветов гексонов и уровней.

**c. Что не успел реализовать:**
Не успел, но планировал реализовать:
- **Object Pool** для создания и удаления гексонов;  
- **DOTween** анимации для собирания стеков;  
- звуковое сопровождение;  
- добавление конфигурации бустеров в **ScriptableObject** уровней;
- Поиск наилучшего перемещения когда доступно несколько ходов.
